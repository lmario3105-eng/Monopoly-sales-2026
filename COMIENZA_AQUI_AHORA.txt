╔══════════════════════════════════════════════════════════════════╗
║                     ¿PROBLEMAS?                                 ║
╚══════════════════════════════════════════════════════════════════╝

❓ Bot no recibe mensajes
   → Abre: /api/whatsapp/auto-sync
   → Verifica: Que todas las variables estén en "ok"
   → Comprueba: Que agregaste las 4 variables

❓ Error en Meta al configurar webhook
   → Token debe ser exactamente: monopolio_sales_2025
   → URL debe ser: https://[tu-proyecto].vercel.app/api/whatsapp
   → Verifica: Que esté publicado en Vercel

❓ Respuestas lentas del bot
   → En Vercel: Revisa los logs de la función
   → Comprueba: Que Gemini API Key esté correcta
   → Ajusta: La logitud de los prompts

❓ Base de datos vacía
   → Es normal: Los datos se guardan en memoria
   → Para persistencia: Conecta a Supabase o Firestore
   → Ejemplos: Ver lib/database.ts
