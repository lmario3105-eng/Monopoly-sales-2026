INSTRUCCIONES RÁPIDAS (5 MINUTOS):
═══════════════════════════════════════════════════════════════════════════════

PASO 1: Agregar tu clave de Jimmy Night a Vercel
   ├─ Ve a: https://vercel.com → Tu Proyecto → Settings
   ├─ Section: "Environment Variables"
   ├─ Add: Name = "JIMMY_NIGHT_API_KEY"
   ├─ Value = [Tu clave de Jimmy Night]
   └─ Click: "Save"

PASO 2: Hacer push de la rama a GitHub (En tu terminal)
   ├─ git checkout main
   ├─ git pull origin main
   ├─ git checkout -b gift-hood
   ├─ git add .
   ├─ git commit -m "feat: Add Gift Hood with Jimmy Night API"
   └─ git push origin gift-hood

PASO 3: Redeploy en Vercel
   ├─ Ve a Vercel
   ├─ Tu proyecto se redesplegará automáticamente
   └─ Espera a que termine

PASO 4: Probar Gift Hood
   ├─ Abre tu aplicación
   ├─ Ve a http://tu-app.vercel.app/api/keys?service=jimmy-night
   ├─ Deberías ver: "status": "active"
   └─ ¡Listo! Gift Hood está funcionando
