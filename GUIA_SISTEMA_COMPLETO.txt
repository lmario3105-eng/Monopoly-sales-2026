2. CONEXIÓN WHATSAPP CON VERIFICACIÓN
──────────────────────────────────────

✅ CARACTERÍSTICAS:
• Registro con número telefónico
• Verificación por código (OTP) de 6 dígitos
• Sincronización bidireccional web ↔ WhatsApp
• Historial unificado en ambas plataformas
• Notificaciones en tiempo real

📍 ARCHIVOS CREADOS:
/lib/whatsapp-types.ts - Tipos de datos y almacenamiento
/app/api/whatsapp/register/route.ts - API de registro y verificación
/components/chat/WhatsAppConnectModal.tsx - UI de conexión

🚀 CÓMO CONECTAR WHATSAPP:
1. En chat, haz click en botón "WhatsApp"
2. Ingresa tu número: +57 300 1234567
3. Recibirás código en WhatsApp (en dev se muestra en console)
4. Ingresa el código de 6 dígitos
5. ¡Listo! Sincronización activa

📱 EN PRODUCCIÓN:
Para conectar WhatsApp real, necesitas:
- WHATSAPP_PHONE_NUMBER_ID (Meta)
- WHATSAPP_BUSINESS_ACCOUNT_ID (Meta)
- WHATSAPP_ACCESS_TOKEN (Meta)

Los códigos se enviarían por SMS/WhatsApp real
